Lockdown Mode

When enabled, all non-admin users are logged out and cannot access the wiki. A lockdown page is shown in place of the normal login screen.
This message is shown to visitors while lockdown mode is active.

Session Limit

When enabled, signing in from a new device or tab invalidates any previous session for that user. This prevents users from being logged in on multiple devices or browser tabs simultaneously.

Page Reservations

When enabled, editors can reserve pages before making destructive changes. Non-admins are blocked from editing reserved pages, while administrators can still override reservations with a clear warning.
How long a page stays reserved before the reservation expires automatically. Default: 48 hours.
How long the reserving user must wait before reserving the same page again after a release or expiry. Default: 24 hours.
Default reservation quota for users who have not received an approved quota increase. Default: 5 pages.
